# Environment Setup

You will need to make sure you have these packages installed:

* foundry
* lcov

Then you need to ensure your environment has these variables set:

```
MAINNET_RPC_URL=
```


# Project Initialization

```
foundryup
```

# Running a build

```
forge build
```

# Running the tests

```
forge test
```

# Generating a coverage report

```
npm run coverage
```
